# Тестовое задание Only

## О проекте

- Проект выполненен на чистом HTML, CSS, JS
- Слайдер выполнен с помощью библиотеки [Splide](https://splidejs.com/)
- Для полноэкранного просмотра изображений использовался [Fancybox](https://fancyapps.com/fancybox/)
- Код написан с использованием принципов БЭМ и PixelPerfect
